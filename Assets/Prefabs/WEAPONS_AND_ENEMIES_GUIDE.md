## Настройка в WaveSpawner

### Что нужно сделать:

1. **Создать префабы врагов** (если ещё не созданы)
   - Скопировать существующий префаб врага из `Assets/Prefabs/Enemy/Enemy1/`
   - Создать 9 вариантов для каждого типа
   - Настроить визуал (цвет через `tintColor` в `.asset`)

2. **Назначить префабы в `.asset` файлы**
   - Открыть каждый `.asset` файл врага
   - В поле `prefab` перетащить соответствующий префаб

3. **Добавить врагов в WaveSpawner**
   - Открыть сцену с `WaveSpawner`
   - В `Enemy Pool` добавить все 10 типов врагов
   - Настроить `minWave` и `weight` для каждого

### Пример конфигурации Enemy Pool:

```
Enemy Pool (10 записей):
┌─────────────────┬─────────┬────────┐
│ Enemy Data      │ minWave │ weight │
├─────────────────┼─────────┼────────┤
│ RiflemanBasic   │    1    │  1.0   │
│ KamikazeRunner  │    1    │  0.3   │
│ MeleeRusher     │    6    │  0.5   │
│ HeavyGunner     │   11    │  0.4   │
│ SniperEnemy     │   11    │  0.2   │
│ ShieldBearer    │   21    │  0.3   │
│ FastAssault     │   21    │  0.4   │
│ BruiserMelee    │   41    │  0.3   │
│ BossJuggernaut  │    5    │  0.0*  │
│ BossWarlock     │    5    │  0.0*  │
└─────────────────┴─────────┴────────┘

* Боссы спавнятся автоматически через bossEveryNWaves
```

### Параметры WaveSpawner:
- **bossEveryNWaves**: 5 (боссы каждые 5 волн)
- **Spawn Points**: Расставить точки спавна по карте
- **spawnJitterRadius**: 1.5 (разброс спавна)
